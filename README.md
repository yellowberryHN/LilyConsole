# LilyConsole
A .NET Framework 4.5.2 library to interface with the various hardware
of a WACCA cabinet.

### Support Status
- [X] Console Touch (COM3/4)
- [X] Console Lights (FTDI)
- [X] Panel VFD (COM2)
- [ ] Panel Lights (IO4)
- [ ] Volume/Test/Service Buttons (IO4)
- [ ] Coin Sensor (IO4)
- [ ] Coin Blocker (IO4)
- [ ] Card Reader (COM1)
